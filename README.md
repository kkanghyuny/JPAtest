## 주요 기능

- 사용자 인증 및 권한 관리
- 행성(챌린지) 생성 및 관리
- 챌린지 참여 및 진행 상황 추적
- 인증 사진 업로드 및 검증
- 사용자 통계 및 성과 분석

## 개발 환경

- Backend
  - Java 17
  - Spring Boot 2.5.x
  - Gradle
- Frontend
  - React 17.x
  - Node.js 14.x
- Database
  - MySQL 8.0
  - Redis 6.x

## 시작하기

```sh
# 백엔드 실행
$ ./gradlew bootRun

# 프론트엔드 실행
$ cd frontend
$ npm install
$ npm start
```
